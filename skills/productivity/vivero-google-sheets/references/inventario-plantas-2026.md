## Notas para importación a Portal

El INVENTARIO solo tiene **PLANTA** y **STOCK** como columnas activas. Para importar al portal vía `stupefy`, necesitas datos adicionales:

| Dato requerido | Fuente probable |
|----------------|-----------------|
| Código (PL-*) | Generar automáticamente: `PL-{SLUGIFY(PLANTA)}` |
| Nombre | Campo PLANTA (ya en mayúsculas) |
| Lista de precio | Elegir: `COST_HACIENDA` o `COST_SARA` |
| Precio USD | Solicitar al usuario o dejar vacío |
| UOM | Usualmente `"U"` (Unidad) |
| Tax category | `097722df-e70c-4380-9372-a863c67c2bca` (EXCENTO) |
